**Test ID:** TCA001

**Test Title:** Verify About section static elements display correctly

**Description:** Verify About section heading and description paragraphs display correctly on page load

**Preconditions:**
- Website https://tracynjoroge.vercel.app/ is open in a desktop browser
- Internet connection is available
- User is currently viewing the About section

**Steps:**
1. Check the About heading is visible
2. Check all three description paragraphs are fully visible

**Expected Result:** 
- About heading is fully visible and readable against the dark background
- All three description paragraphs are fully visible, readable, and not cut off

**Post Condition:** User is now viewing the About section

**Test Type:** Positive

**Status:**

**Test ID:** TCA002

**Test Title:** Verify profile image changes to color on hover

**Description:** Verify the profile image in the About section changes from black and white to color on hover

**Preconditions:**
- Website https://tracynjoroge.vercel.app/ is open in a desktop browser
- Internet connection is available
- User is currently viewing the About section

**Steps:**
1. Hover on the Profile image 
2. Observe what happens

**Expected Result:** 
- Profile image changes from black and white to full color on hover 
- Image shadow appears on the bottom left and bottom of the image on hover

**Post Condition:** User is now viewing the About section. Profile image is displayed in color with shadow visible

**Test Type:** Positive
